1. Install node modules from https://nodejs.org/en/download
2. Install vs code from  https://code.visualstudio.com/download
3. Go to Windows Powershell, Run it as adminstrator and run this command.
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
4. Go to https://github.com/diviontopp/retro-arcade and install the code.
5. Go to VS code terminal and run this command cd retro-arcade-main.
6. Use the command: npm install
7. Then use the command:  npm run dev
